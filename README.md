# Fullstack Task Management App

This is a simple fullstack task management app built with:

- **Backend**: Node.js, Express, TypeScript, SQLite
- **Frontend**: React + Bootstrap

---

## 📁 Project Structure

FullstackTaskApp/
│
├── backend/ # Express + TypeScript backend
│
├── frontend/ # React frontend
│
├── README.md # This file


---

## ⚙️ Requirements

- Node.js (v16 or later)
- npm (comes with Node.js)

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
cd FullstackTaskApp

🧠 Backend Setup
📦 Install dependencies

cd backend
npm install

⚙️ Run the backend

npm run dev

    The backend will start on http://localhost:3001

💻 Frontend Setup
📦 Install dependencies

cd ../frontend
npm install

🚀 Start the frontend

npm start

    The frontend will start on http://localhost:3000

Make sure your backend is also running so the frontend can connect to the API.
📬 API Endpoints

    GET /users

    POST /users

    GET /users/:id/tasks

    POST /users/:id/tasks

✅ Features

    Create users

    Create and view tasks for each user

    Tasks show completion status
